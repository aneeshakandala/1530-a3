Shopping Cart Project

This project serves as a mock-storefront for an imaginary brand! I decided to do a "jewelry store" with different types of jewlery being sold. The project is supposed to allow users to create an account/log into their account, add/delete items to their cart, and checkout. The project is also supposed to show the user's order history. In this project, I used Python, HTML, CSS, and MySQL. It helped serve as an introduction to UI design!
